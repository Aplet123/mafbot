#Mafbot
A mafia bot for Discord.

##Supporting files
`admins.secret.json` - A file with an array of all admin ids as strings.  
`login.secret.json` - A file with an object which has one property `token`, which contains a string with the token of the bot.  
`server.secret.json` - A file with an object which has three properties `channelID`, `serverID`, and `mutedID`, which contains the id of the channel the bot will speak in, the server, and the role to mute people with, respectively.
